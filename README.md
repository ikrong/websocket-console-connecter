# websocket-console-connecter

Use websocket and browser console achieve an convenient debug tool!

# Use
```
npm install -g w2c
w2c serve
```

insert a js label to your html files

```
<doctype html>
<html>
	<head>
		<script src="http://This_is_your_ipaddress:port/console.js"></script>
		........
	</head>
	<body></body>
</html>
```
